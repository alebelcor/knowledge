# [Tailwind CSS](https://tailwindcss.com/)

When extending colors, the `default` value equals a generated class name with no suffix:

```javascript
colors: {
  blue: {
    DEFAULT: '#02427e', // -> text-blue
    light: '#00eff9',   // -> text-blue-light
  }
}
```
