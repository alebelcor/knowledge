# macOS

Unload a user agent:

```bash
launchctl unload <.plist file path>
```

Or just use [LaunchControl](https://www.soma-zone.com/LaunchControl/)

Remove a user agent:

```bash
trash <.plist file path>
```

This assumes the [trash](https://github.com/sindresorhus/trash-cli) command exists.
